# EasyCVMaker 🚀
### منصة صياغة السيرة الذاتية الاحترافية والذكية مجاناً

منصة ويب متكاملة وعصرية تتيح للمستخدمين إنشاء وإدارة وسحب وتخصيص السير الذاتية باللغتين العربية والإنجليزية بأسلوب عرض تفاعلي وتنسيقات عالمية متوافقة بالكامل مع خوارزميات الفرز الآلي للتوظيف (ATS).

---

## 🌟 ميزات المنصة (Core Features)

* **لوحة التحكم المهنية (Professional Dashboard):** تصميم مستوحى من لوحات "بانتو" المستقبلية الداكنة، تتيح إدارة كاملة لملفات السيرة الذاتية (الإنشاء المباشر، التكرار، التعديل أو الحذف الآمن).
* **قوالب عالمية معتمدة (Standard Templates):** قوالب عصرية ومتنوعة (برلين المهني التنفيذي، بانتو المستقبلي العصري، وغيرها) مصممة لتجذب مدراء التوظيف.
* **تحسين محركات البحث والدعم التقني (SEO & Sitemap Support):** توليد فوري وتحميل لملفي `sitemap.xml` و `robots.txt` لتسهيل أرشفة موقعك بمرتبة متقدمة على Google.
* **دعم المدونة ومساهمات الزوار (Community Blog):** إمكانية اقتراح مقالات مهنية من قبل المرشحين ومراجعتها واعتمادها من قِبل المشرفين فوراً.
* **تحسين وتحميل محلي (Offline Saving & JSON Export):** إمكانية استيراد وتصدير بيانات سيرة ذاتية كاملة بصيغة ملفات JSON لضمان المزامنة وحفظ نسخة احتياطية من سيرتك.
* **تحسينات الأداء الفوري (Performance Optimization):** سرعة تحميل فائقة عبر استخدام ميزة تقسيم الملفات المصدرية المتقدمة (Code-Splitting via Lazy Loading) في React + Vite وتخصيص الكتل البرمجية لتقليل فترات انتظار المستخدمين.

---

## 🛠️ التقنيات المستخدمة (Tech Stack)

* **React 18 + TypeScript:** لبناء واجهات تفاعلية متكاملة ومتوافقة مع شروط الأمان البرمجي وتدقيق المجموعات.
* **Vite:** كمحرك تجميع وتطوير فائق السرعة.
* **Tailwind CSS:** لتنسيقات وتصاميم فخمة وعصرية متوافقة مع جميع أحجام الشاشات.
* **Node.js + Express:** خادم ويب متكامل للإنتاج وتقديم الميزات البرمجية (API Routes).
* **Lucide React:** لمجموعة الأيقونات العصرية والمتناسقة.

## 🚀 طريقة التشغيل والتركيب (Setup & Local Execution)

لتشغيل المنصة محلياً على جهازك، يرجى اتباع الخطوات التالية:

### 1. تثبيت الحزم والمكونات البرمجية:
```bash
npm install
```

### 2. تشغيل بيئة التطوير المحلية (Local Development):
```bash
npm run dev
```
سيتم فتح التطبيق على الرابط المحلي: `http://localhost:3000`

### 3. بناء تطبيق الإنتاج (Production Build):
```bash
npm run build
```

### 4. بدء تشغيل الخادم في بيئة الإنتاج:
```bash
npm start
```
